## 📚 KNOWLEDGE BASE

### 1. Quick Reference Formulas
* **SEO Success Formula**: `Good SEO = Technical Foundation + Quality Content + Authority + User Experience + Freshness`
* **AI Citation Formula (GEO)**: `Good GEO = Expertise + Authority + Trustworthiness + Structured Data + Original Insights`

### 2. Core Web Vitals Targets
| Metric | Good | Poor | How to Fix |
|--------|------|------|-----------|
| **LCP** (Largest Contentful Paint) | < 2.5s | > 4.0s | Optimize images, CDN, caching |
| **INP** (Interaction to Next Paint) | < 200ms | > 500ms | Reduce JavaScript, optimize event handlers |
| **CLS** (Cumulative Layout Shift) | < 0.1 | > 0.25 | Reserve space for images, ads |

### 3. E-E-A-T Framework
* **Experience (E)**: First-hand product/service usage, real-world examples, case studies, personal anecdotes, customer testimonials.
* **Expertise (E)**: Author credentials, technical certifications, years of experience, educational background.
* **Authoritativeness (A)**: Backlinks from reputable sources, media mentions, industry awards, expert endorsements, brand reputation.
* **Trustworthiness (T)**: HTTPS security, clear contact info, privacy policy, transparent pricing, customer support.

### 4. Technical SEO Checklist
* **On-Page**: Title tag (50-60 chars), Meta description (150-160 chars), H1-H6 logical hierarchy, URL structure (clean, keyword-rich), Internal linking, Image alt text, Schema markup (JSON-LD).
* **Technical Foundation**: XML sitemap in GSC, robots.txt configured, Canonical tags, HTTPS site-wide, Mobile-friendly responsive design, Structured data validated.
* **Performance**: Page speed < 3s, Optimized/compressed images, Minified CSS/JS, Browser caching, CDN for assets, Lazy loading for below-fold.

### 5. GEO (Generative Engine Optimization)
**What AI Cites & Why:**
* *Original Statistics*: Unique, verifiable data
* *Expert Quotes*: Authority and credibility
* *Clear Definitions*: Extractable, structured info
* *Step-by-Step Guides*: Actionable, practical content
* *Comparison Tables*: Organized, comparative data
* *FAQ Sections*: Direct question-answer format
* *Source Attribution*: Citable, traceable sources

**GEO Best Practices Checklist:**
- [ ] FAQ sections with structured Q&A
- [ ] Author bylines with credentials
- [ ] Publication dates and "last updated"
- [ ] Statistics with clear sources
- [ ] Expert quotes attributed properly
- [ ] Clear, concise definitions & Comprehensive glossary
- [ ] Original research and data

### 6. SEO vs GEO Comparison
| Aspect | Traditional SEO | GEO |
|--------|-----------------|-----|
| **Primary Goal** | Rank #1 in Google | Be cited in AI responses |
| **Platform** | Google, Bing | ChatGPT, Claude, Perplexity |
| **Key Metrics** | Rankings, CTR, traffic | Citation rate, AI mentions |
| **Focus Areas** | Keywords, backlinks | Entities, authority, freshness |
| **Content Style** | Keyword-optimized | Entity-rich, comprehensive |
| **Updates** | Periodic algo updates | Continuous learning |

### 7. Common Mistakes to Avoid
* **Content**: Keyword stuffing, thin/duplicate content, missing meta descriptions, poor heading structure.
* **Technical**: Slow page speed, 404 errors, redirect chains, missing alt text, blocked in robots.txt.
* **UX**: Intrusive interstitials, non-mobile-friendly, poor navigation, annoying pop-ups.

### 8. Schema Markup Types (Use Cases)
* `Article`: Blog posts, news
* `Product`: E-commerce
* `FAQ`: Frequently asked questions
* `HowTo`: Step-by-step guides
* `Review`: Product/service reviews
* `LocalBusiness` / `Organization` / `Person`: Entity info
* `Event` / `Course`: Time-based or educational content
